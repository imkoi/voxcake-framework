# VoxCake.Framework
VoxCake.Framework - IoC for unity engine.

## Getting Started
1. Get latest [release](https://github.com/imkoi/voxcake-framework/releases/tag/1.0) or built it yourself from source.
2. Add reference to assembly in your project.
3. Use it!
